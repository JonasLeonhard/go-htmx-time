# About
a timetracking webapp using go and htmx

Features:

- Track your time
- The time is stored in a local sqlite file

# Building tailwind styles for templates
```bash
just css
```

# Rebuilding the ent db schema
```bash
just ent
```

# Running in dev mode & reload after making changes
```bash
just dev
```

# Running app
```bash
just run
```
